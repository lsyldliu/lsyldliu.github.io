Java 句柄

Share

　　这几天在学习Java，一直在看think in Java这本书，这本书确实不错，让我对面向对象的编程有了更深一层的理解。书里面一直在强调“句柄”这个概念，之前学习C++的之前，并没有去理解这个概念，故写下对它的理解。

1.数组句柄

int[] a1; 
这里创建了一个句柄a1，a1指向一个数组，但并没有给数组分配存储空间，所以数组实际上并不存在。

当令

a1 = new int[10]；

　　此时new int[10]则创建了拥有10个元素的整形数组对象，并令句柄a1指向该数组的首地址，此时可以按操作数组的方式对a1进行操作。

　　当然句柄也可以进行复制操作，这里在创建另外一个数组句柄，前提是要对a1进行初始化。

int[] a1 = { 1, 2, 3, 4, 5 };
int[] a2;
a2 = a1;

　　下面以一段可用的代码来演示一下句柄复制的过程。

public class Arrays {
  public static void main(String[] args) {
      int[] a1 = { 1, 2, 3, 4, 5 };
      int[] a2;
      a2 = a1;
      for(int i = 0; i < a2.length; i++)
      a2[i]++;
      for(int i = 0; i < a1.length; i++)
      prt("a1[" + i + "] = " + a1[i]);
      }

      System.out.println(s);
    }
} ///

2.对象句柄

　　　首先创建一个类Number,

class Number {
    int i;
}

然后创建该类的两个实例

Number n1 = new Number();
Number n2 = new Number();

n1和n2为该类的两个句柄，这两个句柄分别指向该类的两个对象，这里通过new Number()创建类的对象，当然和数组句柄一样，对象句柄也能进行复制，

n2 = n1; 这里令句柄n2指向n1的对象，改变n1的同时也会改变n2,n2的对象则会被垃圾收集器回收

同样这里以实例来演示这种变化

class Number {
       int i;
} 
public class Assignment {
     public static void main(String[] args) {
        Number n1 = new Number();
        Number n2 = new Number();
        n1.i = 9;
        n2.i = 47;
        System.out.println("1: n1.i: " + n1.i +", n2.i: " + n2.i);
        n1 = n2;
        System.out.println("2: n1.i: " + n1.i +", n2.i: " + n2.i);
        n1.i = 27;
        System.out.println("3: n1.i: " + n1.i +", n2.i: " + n2.i);
         }
   }

输出结果如下：

1: n1.i: 9, n2.i: 47
2: n1.i: 47, n2.i: 47
3: n1.i: 27, n2.i: 27
